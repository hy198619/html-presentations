Thariq 的洞察

Markdown 的黄金时代正在过去

Before · 人在写

手敲方便的格式

  • 人自己写文档、做笔记
  • 几个符号就能搞定排版
  • 纯文本好编辑、好迁移
After · AI 在写

不再需要手敲了

  • 告诉 AI 你要什么，AI 帮你写
  • Markdown 最大的优势不存在了
  • 格式只是容器，不是门槛

让 AI 输出 HTML，四个你马上能感受到的好处

不只是排版

01
信息密度
颜色 / 图表 / 流程图 / 标签页 / 时间线 / 幻灯片——Markdown 用文字描述照片，HTML 直接把照片给你看

03

双击即开，上传生成链接，手机电脑都能看——不需要对方装任何软件
04
交互
滑块 / 按钮 / 选项——文档从"只读"变成"可操作"，最让人兴奋的一点

我的区分 · My Take
07 / 10
我自己琢磨的区分

存 vs 输出，是两件事

存给自己看

→ Markdown

  • 笔记、知识库、个人想法
  • 纯文本好搜索
  • 不怕工具换代
要给别人看

→ HTML

  • 方案、报告、调研结果
  • 浏览器打开就能看
  • 手机上也正常显示
你不需要学任何 HTML 知识。
直接告诉 AI："帮我生成一个 HTML 文件"就行。
难点不在于语法，而在于你想清楚——这个文件要做什么、给谁看。
